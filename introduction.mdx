## What is Dev0?

Dev0 is an **agentic engineering platform** inspired by the Ralph loop: let a code agent run until the task is actually done, but keep it contained, auditable, and reviewable.

You provide a product idea. Dev0:

1. Turns your idea into a structured project with a name, summary, and tech stack
2. Generates a phased, dependency-aware task plan using Gemini 3 Pro
3. Executes each task inside an isolated E2B sandbox running Gemini CLI
4. Streams live logs back to your browser in real time
5. Ships every completed task as a GitHub pull request — ready for your review

Engineers stay in control as technical leads. Dev0 handles the execution.

## Use cases

- **Solo engineers** — ship features by delegating tasks to agents while you review
- **Startups** — reduce build time with PR-first automation and a visible audit trail
- **Teams** — keep human review as the control plane for all AI-generated code
- **Hackathon teams** — turn ideas into working code with live progress tracking
